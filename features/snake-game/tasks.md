## 详细任务说明

### T-01: 创建 HTML 骨架结构

**目标**: 创建基础的 HTML 页面结构

**实施步骤**:
1. 创建 `index.html` 文件
2. 添加 HTML5 基本结构（DOCTYPE, html, head, body）
3. 添加页面标题 "贪食蛇游戏"
4. 添加游戏区域容器（标题、得分显示、Canvas）
5. 添加操作说明文字
6. 添加内联 CSS 样式

**验收标准**:
- [ ] HTML 文件可以在浏览器中打开
- [ ] 页面显示标题和游戏区域
- [ ] 样式正常、布局合理

**输出文件**: `features/snake-game/index.html`

---

### T-02: 实现 Canvas 初始化

**目标**: 初始化 Canvas 画布并准备渲染环境

**实施步骤**:
1. 在 HTML 中添加 `<canvas>` 元素（400x400）
2. 在 JavaScript 中获取 Canvas 上下文
3. 设置画布背景色
4. 实现网格绘制函数（可选，便于调试）

**验收标准**:
- [ ] Canvas 元素正确显示
- [ ] 画布大小为 400x400 像素
- [ ] 背景色为白色

**涉及文件**: `index.html`

---

### T-03: 实现游戏数据结构

**目标**: 按照设计文档定义游戏配置和状态

**实施步骤**:
1. 定义 `config` 对象（网格大小、初始速度等）
2. 定义 `gameState` 对象（蛇、食物、得分等）
3. 实现游戏初始化函数 `initGame()`
4. 实现重置游戏函数 `resetGame()`

**验收标准**:
- [ ] config 对象包含所有必要配置
- [ ] gameState 初始值正确
- [ ] initGame() 能正确初始化游戏
- [ ] resetGame() 能重置所有状态

**涉及文件**: `index.html`

---

### T-04: 实现蛇的移动逻辑

**目标**: 实现蛇的移动和键盘控制

**实施步骤**:
1. 实现键盘事件监听器
2. 实现方向控制逻辑（防止直接掉头）
3. 实现蛇身移动算法
4. 实现游戏主循环 `gameLoop()`
5. 实现蛇的渲染函数

**验收标准**:
- [ ] 按方向键能控制蛇的移动方向
- [ ] 蛇按照正确方向移动
- [ ] 不能直接掉头（向上时按下键无效）
- [ ] 画面流畅更新

**涉及文件**: `index.html`

---

### T-05: 实现碰撞检测

**目标**: 实现撞墙和撞自己的检测

**实施步骤**:
1. 实现撞墙检测逻辑
2. 实现撞自己检测逻辑
3. 在游戏循环中集成碰撞检测
4. 触发游戏结束状态

**验收标准**:
- [ ] 蛇撞墙时游戏结束
- [ ] 蛇撞到自己时游戏结束
- [ ] 碰撞后游戏停止响应

**涉及文件**: `index.html`

---

### T-06: 实现食物系统

**目标**: 实现食物生成和吃食物逻辑

**实施步骤**:
1. 实现随机生成食物函数
2. 确保食物不生成在蛇身上
3. 实现吃食物检测
4. 吃到食物后蛇身增长
5. 重新生成新食物

**验收标准**:
- [ ] 食物随机出现在画布上
- [ ] 食物不在蛇身上
- [ ] 蛇吃到食物后身体变长
- [ ] 吃到食物后生成新食物

**涉及文件**: `index.html`

---

### T-07: 实现计分系统

**目标**: 显示和更新得分

**实施步骤**:
1. 在 HTML 中添加得分显示元素
2. 初始化得分为 0
3. 吃到食物后得分 +10
4. 更新得分显示

**验收标准**:
- [ ] 初始得分显示为 0
- [ ] 吃到食物后得分增加 10
- [ ] 得分实时更新显示

**涉及文件**: `index.html`

---

### T-08: 实现速度控制

**目标**: 随着得分增加提高游戏速度

**实施步骤**:
1. 在 config 中定义速度增量
2. 每次吃食物后减少帧间隔时间
3. 设置最小速度限制
4. 在 setTimeout 中使用动态速度

**验收标准**:
- [ ] 初始速度为 150ms/帧
- [ ] 每吃一个食物速度提升 5ms
- [ ] 速度不会低于 50ms/帧
- [ ] 速度提升有明显的游戏体验变化

**涉及文件**: `index.html`

---

### T-09: 实现游戏结束处理

**目标**: 显示游戏结束界面和最终得分

**实施步骤**:
1. 实现游戏结束函数 `gameOver()`
2. 在画布上绘制"游戏结束"文字
3. 显示最终得分
4. 停止游戏循环

**验收标准**:
- [ ] 碰撞后显示"游戏结束"
- [ ] 显示最终得分
- [ ] 游戏循环停止

**涉及文件**: `index.html`

---

### T-10: 实现重新开始功能

**目标**: 游戏结束后可以重新开始

**实施步骤**:
1. 添加"重新开始"按钮
2. 实现重新开始点击事件
3. 调用 `resetGame()` 重置状态
4. 重新启动游戏循环

**验收标准**:
- [ ] 游戏结束界面显示"重新开始"按钮
- [ ] 点击按钮后游戏重置
- [ ] 得分清零、蛇回到初始状态
- [ ] 游戏重新开始运行

**涉及文件**: `index.html`
