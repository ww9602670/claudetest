# 操作步骤 — 网页版贪食蛇游戏

> 小白版 | feature-id: snake-game

---

## 怎么打开和玩这个游戏？

### 第一步：找到游戏文件

游戏文件在这个位置：
```
features/snake-game/index.html
```

### 第二步：用浏览器打开

你可以用以下方式打开：

**方法1：直接双击**
- 在文件管理器中找到 `index.html`
- 双击文件，它会用默认浏览器打开

**方法2：右键选择浏览器**
- 右键点击 `index.html`
- 选择"打开方式" → 选择你的浏览器（Chrome/Firefox/Edge）

**方法3：拖拽到浏览器**
- 打开你的浏览器
- 把 `index.html` 文件拖到浏览器窗口中

### 第三步：开始游戏

游戏打开后会自动开始，你会看到：
- 一个白色的方块（游戏区域）
- 一条绿色的蛇在移动
- 一个红色的食物
- 显示"得分: 0"

---

## 怎么操作游戏？

### 基本控制

**控制方向**：
- 按键盘上的 **↑** (向上箭头) - 蛇往上走
- 按键盘上的 **↓** (向下箭头) - 蛇往下走
- 按键盘上的 **←** (向左箭头) - 蛇往左走
- 按键盘上的 **→** (向右箭头) - 蛇往右走

**注意事项**：
- 蛇会一直移动，不能停止
- 你只能改变方向，不能让蛇停下
- 不能直接掉头（比如蛇正在往上走，你不能马上让它往下走）

### 游戏规则

**得分**：
- 吃到一个食物，得分增加 10 分
- 吃到食物后，蛇会变长一节

**游戏结束**：
- 蛇头撞到墙壁 - 游戏结束
- 蛇头撞到自己的身体 - 游戏结束

**重新开始**：
- 游戏结束后，点击"重新开始"按钮
- 游戏会重置，得分为 0，蛇回到初始状态

## 新手建议

### 初级策略
1. **不要急** - 游戏开始时速度较慢，先熟悉控制
2. **规划路线** - 提前想好怎么去吃食物
3. **避免角落** - 尽量不要让蛇在角落里，容易卡住

### 进阶技巧
1. **沿着边缘走** - 在游戏区域边缘移动更安全
2. **预留空间** - 给蛇的身体留足够的移动空间
3. **适应速度** - 随着得分增加，速度会变快，要提前反应

### 常见错误
- ❌ 急转弯导致撞到自己
- ❌ 在狭小空间里转圈
- ❌ 没有提前规划路线

## 如果游戏有问题怎么办？

### 问题1：游戏画面不显示
**可能原因**：
- 浏览器太旧，不支持 Canvas

**解决方法**：
- 更新浏览器到最新版本
- 使用 Chrome、Firefox 或 Edge 浏览器

### 问题2：按方向键没反应
**可能原因**：
- 游戏窗口没有获得焦点

**解决方法**：
- 用鼠标点击一下游戏画面
- 然后再按方向键

### 问题3：游戏太卡
**可能原因**：
- 电脑性能较低
- 浏览器开了太多标签页

**解决方法**：
- 关闭其他浏览器标签页
- 关闭其他占用电脑的程序

### 问题4：蛇不听使唤
**可能原因**：
- 快速按键导致冲突

**解决方法**：
- 慢慢按，不要连续快速按方向键
- 提前规划好路线，避免频繁改变方向
