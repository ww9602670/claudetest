# 网页版贪食蛇游戏

一个经典的贪食蛇游戏，用于验证 spec 驱动开发机制。

---

## 游戏简介

这是一个使用纯 HTML、CSS 和 JavaScript 实现的经典贪食蛇游戏。玩家通过键盘方向键控制蛇的移动，吃到食物后蛇身变长并得分，撞墙或撞到自己则游戏结束。

**开发目的**：作为 04 阶段的首个正式实施对象，用于验证 spec 驱动开发、非程序员说明同步产出、自动开发/修复、evidence 与复审机制是否完整可行。

---

## 快速开始

### 1. 打开游戏

在文件管理器中找到 `index.html` 文件，双击即可在浏览器中打开。

### 2. 开始游戏

游戏会自动开始，使用键盘方向键控制蛇的移动：
- **↑** 向上移动
- **↓** 向下移动
- **←** 向左移动
- **→** 向右移动

### 3. 游戏规则

- 吃到红色的食物可以得分并变长
- 每吃一个食物得 10 分
- 随着得分增加，蛇的速度会逐渐加快
- 撞墙或撞到自己游戏结束
- 游戏结束后点击"重新开始"按钮可以重新开始

## 游戏操作

### 基本控制
| 按键 | 功能 |
|------|------|
| ↑ | 蛇向上移动 |
| ↓ | 蛇向下移动 |
| ← | 蛇向左移动 |
| → | 蛇向右移动 |

### 注意事项
- 蛇会一直移动，不能停止
- 不能直接掉头（例如向上时不能直接向下）
- 速度会随着得分增加而加快

## 开发信息

### 开发目标
本游戏的主要目标是验证一套软件开发方法，而非提供完美的游戏体验。因此：
- 优先保证开发流程的完整性和可追溯性
- 游戏功能完整度是次要目标
- 后续可以基于此版本进行功能扩展

### 开发流程
1. **Planning 阶段**：编写需求、设计、任务文档
2. **Implementing 阶段**：实现游戏代码
3. **Verifying 阶段**：测试和验证

### 验证标准
- ✅ Spec 三件套完整且驱动了实际开发
- ✅ 非程序员可读说明已同步产出
- ✅ Required_checks 已建立且可执行
- ✅ Evidence 链完整可追溯
- ✅ 自动修复过程有记录且在边界内

## 常见问题

### Q: 游戏画面不显示？
**A**: 请确保你的浏览器支持 HTML5 Canvas，并更新到最新版本。

### Q: 按方向键没反应？
**A**: 点击一下游戏画面，确保浏览器窗口获得焦点，然后再按方向键。

### Q: 游戏太卡？
**A**: 关闭其他浏览器标签页和占用资源的程序，确保电脑有足够的性能。

### Q: 蛇不听使唤？
**A**: 不要快速连续按方向键，慢慢按，提前规划好路线。
