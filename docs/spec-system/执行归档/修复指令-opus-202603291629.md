## 2. 不要把“真实工程入口”写成未经验证的固定命令集合

你必须把这一块改成：

* **若项目现实中不存在某命令/脚本时的处理方式**

也就是说：

* 非程序员说明是否同步可产出
* 自动修复是否可控
* evidence / 复审链是否成立
