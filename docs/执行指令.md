# 执行指令入口

本项目执行指令已拆分为两份：

1. `docs/执行指令-opus.md`
   - 用于 Opus 4.6
   - 负责审计、计划、交接
   - 只允许生成 `docs/handoff/phase1-plan.json`

2. `docs/执行指令-sonnet.md`
   - 用于 Sonnet 4.6
   - 负责第一阶段最小落地施工
   - 只允许依据交接文件施工，不得进入第二阶段

执行顺序：

1. 先运行 Opus 指令
2. 确认生成 `docs/handoff/phase1-plan.json`
3. 再运行 Sonnet 指令
4. Sonnet 完成后停止，等待审核

旧版“大一统执行方式”已废弃，不再使用。
